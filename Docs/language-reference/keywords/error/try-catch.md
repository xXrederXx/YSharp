# TRY-CATCH

## Description

**IMPORTANT** this feature is not ready yet. using it could break your code or cause unexpected behaviour.

The Try-Catch keywords can be used to handle exceptions. when an exception occurs inside a try block, the code will continue in the catch where you can handle it. This is usefull to prevent crashing the application.
